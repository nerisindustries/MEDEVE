Exerecício 3 explicção de proque organizar o projeto em pastas é importante.
Em projetos pequenos é tranqilo organizar tudo em um único arquivo, porém em projetos maiores, isso causa: código confuso,Dificuldade de manutenção, Erros frequentes e dificuldade de trabalhar em equipes.


Exerecício 3 explicção das pastas......
data:é uma pasta que simula, ou melhor, tem a função de um banco de dados, usando arrays para simular tabelas,além de ser muito usado na por iniciantes da progrmação facilita mudar para o Mysql.
includes:é onde colocamos a maior parte do nosso código ,é a base do nosso prjeto, nela estão os aquivos php que contem html ou link paras bootstrap por exemplo.
pages:É onde colocaremos a home page do nosso projeto, no caso a tela inical após o login.
